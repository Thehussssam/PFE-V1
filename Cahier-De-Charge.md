# Cahier des Charges : JobConnect — Plateforme de Recrutement Web

## 1. Introduction

Dans le cadre de notre projet de développement web, nous avons choisi de réaliser une plateforme nommée **JobConnect**.

Cette plateforme a pour objectif de moderniser le processus de recrutement en facilitant la communication entre les candidats et les entreprises.

Aujourd’hui, plusieurs entreprises utilisent encore des méthodes classiques de recrutement basées sur les emails, les réseaux sociaux ou les échanges manuels, ce qui peut compliquer la gestion des candidatures et le suivi des recrutements.

Le projet **JobConnect** propose donc une solution web moderne permettant de centraliser les offres d’emploi, les candidatures et les échanges dans une seule plateforme intuitive et professionnelle.

L’application permettra principalement :

- la publication des offres d’emploi,
- la recherche d’opportunités professionnelles,
- la gestion des candidatures,
- le suivi des recrutements,
- la communication entre candidats et recruteurs.

---

# 2. Objectifs du Projet

Les principaux objectifs sont :

- Digitaliser le processus de recrutement.
- Simplifier la recherche d’emploi pour les candidats.
- Permettre aux entreprises de gérer leurs offres facilement.
- Centraliser les candidatures dans une seule plateforme.
- Offrir une expérience utilisateur moderne et intuitive.

# 4. Utilisateurs du Système

L’application comporte trois types d’utilisateurs :

## Candidat

- Créer un compte
- Se connecter à la plateforme
- Rechercher des offres d’emploi
- Postuler aux offres
- Gérer son profil et son CV
- Consulter ses candidatures

## Recruteur / Entreprise

- Créer un compte entreprise
- Publier des offres d’emploi
- Gérer les annonces publiées
- Consulter les candidatures reçues
- Rechercher des candidats

## Administrateur

- Gérer les utilisateurs
- Gérer les entreprises
- Contrôler les offres publiées
- Consulter les statistiques générales
- Superviser la plateforme

---

# 5. Fonctionnalités Principales

## Gestion des utilisateurs

- Création de compte
- Connexion sécurisée
- Gestion des profils
- Upload du CV

## Gestion des offres d’emploi

- Ajouter une offre d’emploi
- Modifier ou supprimer une offre
- Définir :
  - le poste,
  - le salaire,
  - le type de contrat,
  - la localisation,
  - les compétences demandées

- Recherche et filtrage des offres

## Gestion des candidatures

- Postuler à une offre
- Suivre les candidatures envoyées
- Consulter les candidatures reçues
- Modifier le statut :
  - En attente
  - Acceptée
  - Refusée

## Tableau de bord

### Dashboard candidat

- Nombre de candidatures
- Offres sauvegardées
- Notifications
- Gestion du profil

### Dashboard recruteur

- Nombre d’offres publiées
- Nombre de candidatures reçues
- Gestion des annonces

### Dashboard administrateur

- Gestion globale de la plateforme
- Gestion des utilisateurs
- Statistiques générales
